核心必填变量：

订阅链接：
SUB_URL

账号配置信息 JSON：
USERS_JSON

https://www.json.cn/js-format/

（填入的数据格式参考）：
多账号
[
    {
        "username": "邮箱",
        "password": "密码",
        "serverId": "服务器ID"
    },
    {
        "username": "邮箱",
        "password": "密码",
        "serverId": "服务器ID"
    }
]
单账号
[
    {
        "username": "邮箱",
        "password": "密码",
        "serverId": "服务器ID"
    }
]

Telegram 机器人 Token：
TG_BOT_TOKEN

Telegram 接收频道或个人 ID：
TG_CHAT_ID


可选配置变量：
指定 TG 话题 ID（如果需要发到特定话题）：
TG_THREAD_ID
